# AI Modification Comment Rule

When you modify any file, add the following comment at the very beginning of the file (after any shebang line if present):

- For JavaScript/TypeScript files: `// File modified by Amazon Q`
- For Python files: `# File modified by Amazon Q`
- For CSS/SCSS files: `/* File modified by Amazon Q */`
- For HTML files: `<!-- File modified by Amazon Q -->`

Only add the comment if it is not already present in the file.
